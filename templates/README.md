## 使用说明

1. 选择适合您项目类型的模板
2. 复制内容到项目根目录的 `CLAUDE.md` 文件
3. 根据项目实际情况修改配置
4. 随着项目发展持续更新和完善

## 自定义建议

- 根据团队编码规范调整代码风格部分
- 添加项目特定的命令和工作流程
- 定期更新依赖版本和工具配置
- 记录常见问题和解决方案
